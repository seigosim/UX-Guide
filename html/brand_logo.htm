Usage Guidelines

The Arbonne logo has very specific usage guidelines that must be observed whenever it is used.

  1. Maintain the correct proportions of the logo, do not alter the logo in a way that will distort the shape. You should only scale the vector version, never the web versions.
  2. Reproduce the logo in the official signature color. It is preferred to use the logo on a white background. If you must use it on a dark background, you may do so as long as there is a high contrast between the background and logo. The logo should always be reproduced in a solid color without transparency.
  3. The Arbonne logo cannot be combined with any other text, logos or decoration and must always maintain a clear separation from any surrounding layout elements.
  4. The Arbonne logo represents the Arbonne brand and should never be used in a way that would create confusion with another brand, product or company.
  5. The registered trademark symbol (circled R) must always accompany the logo except in instances where the logo is being used in a very small size.
