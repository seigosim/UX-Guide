Arbonne UX/UI Style Guidelines

This style guide outlines the principles and standards for the creation of web pages for Arbonne. It is intended for use by designers, developers and producers.

The intended use of this document is to establish a consistent quality of presentation, interaction and experiences for fundamental elements across the Arbonne online properties.

Main areas:
  1. High-level wireframes for major page types and the rules for constructing them.
  2. Information on the structure and limitations of each section.
  3. Detailed visual styles and specifications of screen elements.
  4. Technical requirements and standards for Arbonne web pages.

How to use this document
