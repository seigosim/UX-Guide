Banner

The Banner Component can be formed out of a single image, an image background with an open HTML foreground layer, or a solid background color with an open HTML foreground layer.

Example:
Brownie caramels gingerbread wypas wypas pie. Fruitcake cake toffee cake cupcake powder. Topping liquorice faworki tootsie roll jujubes applicake.
VIEW DETAILS
Usage:

The dimensions of the banner are set by the background image size or by specifying a height and width. The height and width can be specified in pixels or percentage.

Elements in the foreground layer can be positioned using absolute positioning or by padding.

Using percentage-based sizes in the foreground layer can allow the banner to act responsively with the page size. Minimum size limits should be specified to avoid having the contents overflow outside the banner container.

Code Sample:
<div class="banner_module">
 <div class="background" style="background-color:#ECE2D8; width:auto; height:auto"><img src='../img
/re9_banner_notext.png' /></div>
 <div class="foreground">
  <div  style="padding:69px 0 0 34px ;width: 320px;">
   <div style="font-size:14px; line-height:18px; color: #777777; background-color: #EFE5DB">Brownie caramels gingerbread wypas wypas pie. Fruitcake cake toffee cake cupcake powder. Topping liquorice faworki tootsie roll jujubes applicake.</div>
   <a href="#" style="display:block; font-size:11px; font-weight: bold; position: relative; margin-top:6px;  background-color: #EFE5DB">VIEW DETAILS <span class="triangle-right green" style=" margin-left: 6px"></span></a>
  </div>
 </div>
</div>
CMS options:
Background Image
[Image Asset] or none
Background Image Alt Text
String (*required if using a background image)
Background Image Link
[URL] or none
Foreground HTML
[HTML Asset] (*optional)
CMS Template:
<div class="banner_module camera_effected">
 <div class="background"><*BGLINK*><*BGIMAGE*><*/BGLINK*></div>
 <div class="foreground">
  <*FGHTML*>
 </div>
</div>


Variables:
Foreground HTML: <*FGHTML*> ("" or HTML Asset)
Background URL: <*BGURL*> (URL)
Background Link Open: <*BGLINK*>  ('<a href="<*BGURL*>">') (optional)
Background Link Close: <*/BGLINK*>  ('</a>') (optional)
Background Image: <*BGIMAGE*>  ("<img src='<*BGIMGSRC*>' alt='<*BGALT*>' />") (optional)
Background Image Source: <*BGIMGSRC*>  (Image Asset)
Background Image Alt Text: <*BGALT*>  (String)
