Layout

Layout styles are used for the "block" elements that have a defined size or position. Using the grid system, you can create multi-column layouts and control how elements flow in the document.

Twelve-column Grid Layout

The grid is divided into twelve equal columns across a containing element. The default behavior of the page types is to have columns that have a minimum and maximum width but can be dynamically sized as the browser window changes.

Because of this flexible layout, try to avoid dividing the available space into too many columns. Try using two or three columns at the most. You can take advantage of unequal width columns by selecting the appropriate column width classes.

All grids start with a "row" element. Within the row, you add the column elements. The example to the right shows two equal columns. The <div> element generally gives you the most flexibility to use the grid system but any block-type element will work. 
Grid Example:
<div class="row">
 <div class="column six">Column 1</div>
 <div class="column six">Column 2</div>
</div>

There are also some special-purpose classes for two and three equal-width column layouts.

Two Across style (class="two_across") This style will give you two equal columns but to allow for the gutter space, adding blocks within the columns with class="inner" will assign the correct padding between columns.

Three Across (class="three_across") You can also have three equal columns using the three_across style. Remember to include inner blocks to allow the padding to apply correctly. 

Two Across Example:
<div class="row">
 <div class="two_across">
  <div class="inner">Column 1</div>
 </div>
 <div class="two_across">
  <div class="inner">Column 2</div>
 </div>
</div>

Padding and Margins

By default, the column classes have no extra space between each other and no extra space within. This allows you to tightly pack blocks however, in most cases, you would like to have some space between elements.

To add space outside a block, use the margin classes. To add space within and element, padding classes are used.

Note: margin and padding is added to the overall width of an element, so if you assign them to a column class, you may find that elements "wrap" to a new line. To avoid this, add a block element inside the column element and apply padding or margin to that.

In the example below, three columns with class="four" should fit across the row equally but since I added "margin-right" on the first two columns, their size has grown and exceeded the total width of the row causing the third column to wrap to a new line.

Example 1:
<div class="row space_before space_after">
  <div class="column four margin-right gray-background bordered">Column 1</div>
  <div class="column four margin-right gray-background bordered">Column 2</div>
  <div class="column four gray-background bordered">Column 3</div>
</div>

Example 2:
<div class="row space_before">
  <div class="column four"><div class="margin-right padded gray-background bordered">Column 1</div></div>
  <div class="column four"><div class="margin-right padded gray-background bordered">Column 2</div></div>
  <div class="column four"><div class="padded gray-background bordered">Column 3</div></div>
</div>

Margin and padding can be applied to all sides of the element or only on specified sides. To add some space above or below a row, use the class="space_before" and class="space_after" on row elements.
