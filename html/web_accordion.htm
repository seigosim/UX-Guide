Accordion

The Accordion component is a variation on the Tab Group component. The "tabs" are stacked vertically with on the titles showing by default. Titles can be clicked to expand its contents which also closes any previously-opened tab.

Properties:
  • Minimum tab: 2
  • Maximum tabs: no maximum
  • One tab can optionally be specifed to be open on page load.
  • In "toggle" mode, more than one tab can be open at a time.

Usage:
  • The accordion is a type of tab group and is built around the unordered list element (<ul>)
  • Add class="accordion" to the tabgroup
  • Each tab must contain one element with class="title"
  • Each tab must contain one element with class="body"
  • To open one tab by default, class="active" to only that <li>.
  • By default, ony one tab can be open at a time. To have multiple tabs open, add class="toggle" to the tabgroup element. Each tab can still be collapsed by clicking its title.
  • The tabgroup element can have multiple classes assigned to it, simply leave a space between the classs names. Example: class="tabgroup accordion toggle" will define an accordion component that has toggled tabs.
Code Sample:



<ul class="tabgroup accordion">
 <li class="active">
  <a href="#" class="title">Item 1</a>
  <div class="body">
   Accordion Content Here
  </div>
 </li>
 <li">
  <a href="#" class="title">Item 2</a>
  <div class="body" style="height:350px">
   Accordion Content Here
  </div>
 </li>
</ul>
