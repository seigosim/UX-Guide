Usage

Start with a <div class="pagination" >

Next add buttons with <div class="pgbtn" >. Inside the button element should be one link element.

Some of the buttons have special properties that are invoked via the following classes:

first
The leftmost block is required to be class="first". By default this will have the double-chevron icon.
last
The leftmost block is required to be class="last". By default this will have the double-chevron icon.
previous
By default this will have the chevron icon.
next
By default this will have the chevron icon.
noicon
Use this class to suppress default icons so you can use your own text.
active
Use this class to indicate the current page number. (Should only appear once)
disabled
Use on any button that should not be clicked in the current context. Example: on page 1, disable the previous button.
large
Makes larger, stand alone button types. No icon by default.
form
A form inside this button will be styled appropriately

You can use the noicon class to suppress the normal default icon and type any text. To have arbitrary text and the icon, you will add <span class="icon"> </span>inside the a element

To specify a different icon than the default, add the class to the span (first, last, next or previous)
