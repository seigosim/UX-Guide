Typography

To keep typography consistent across the web properties, most font parameters are controlled by the site global stylesheet. When building content, you should use HTML tags that represents the semantic intent of the content. This means that you should build structure using headings, subheadings, paragraphs and list elements where appropriate. The global stylesheet will automatically apply the appropriate visual styling.

When you need more control over the styling, you may use the pre-defined typography classes. Avoid adding style delcarations within HTML tags.

Text Styles

These styles deal with text. You can combine more than one.

allcaps
Converts all text within the element to upper-case
text-shadow
Adds a light drop shadow to text
text-inset
Makes text appear indented by adding a slight shadow to the upper left. Use with a background color.
underline
Underline the text
highlight
Yellow highlighter background
nowrap
Text will be forced on one line. This will also force blocks to be wider than specified, so use sparingly!
active
The active nav item. Bolder and usually different background color
error
Use for warnings and notifications
notice
Use for notifications
