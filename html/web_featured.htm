Featured Products

The Featured Products component places 3 or 4 product promo blocks across the page.

Care must be taken to select the titles and description fields so that each product module displays approximately the same amount of copy. When one module contains many more lines of text it will cause the modules to look uneven.

There is a dynamic behavior that attempts to even out the bottom edge of modules, but this can only compensate for a minor difference in height between the modules.

Example:

CMS options:
Module Title
String or "none" (Default = "Featured Products")
Module Link Text
String or "none" (Default = "View All [Brand] Products")
Module Link URL
URL or "none" (Links to product category)
Select 3 products:
Product ID
ProductId
Call To Action Text
String (Default = "View Details")
CMS Template:
Subtemplate = <*MODULESUBTEMPLATE*> :
 <div class="featured_product_module column four" >
  <div class="module_highlight">
   <a href="<*PRODDETAILLINK*>"><img src="<*LARGEIMG*>" alt=""/></a>
   <div class="content_group">
    <h6><a href="<*PRODDETAILLINK*>"><*PRODNAME*></a></h6>
     <*SHORTDESC*>
    <a href="<*PRODDETAILLINK*>" class="call_to_action"><*CTALINKTEXT*></a>

  </div>
 </div>

Subtemplate Variables:
  Call to Action Text: <*CTALINKTEXT*>
 From the database:
  Product Name: <*PRODNAME*>
  Short Description: <*SHORTDESC*>
  Product Large Image: <*LARGEIMG*>
  Product Detail Link: <*PRODDETAILLINK*>
