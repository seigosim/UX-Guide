Web Colors

Colors in the offical Arbonne web palette can be accessed in code by specifying the associated class names. There are separate class names for text color and for background color.

It is not acceptable to use the hex colors directly within code. By using the class name, it allows us to globally assign the color palette.

Brand Colors

Light Gray #F1F1F1

class="light_gray"
class="light_gray_background"

Mid Gray #727272

class="mid_gray"
class="mid_gray_background"

Dark Gray #424242

class="dark_gray"
class="dark_gray_background"

Arbonne Green #417630

class="arbonne_green"
class="arbonne_green_background"

Bright Green #3d9b35

class="bright_green"
class="bright_green_background"
Application Colors

Light Blue #40B4C2

class="light_blue"
class="light_blue_background"

Magenta #C24093

class="magenta"
class="magenta_background"

Default Link Colors

The brand stylesheet specifies the default link color and includes the variations for the link states. It is not acceptable nor necessary to hard code the hex color into child stylesheets as all links will inherit this color by default.

For navigation-style links, it may be desired to force a different color. This is the only case where it is acceptable to override the link color.

Example Link (hover)

Link: #3D9B35
Visited: #3D9B35
Active: #3D9B35
Hover: #417630
