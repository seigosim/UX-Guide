Tooltip

The Tooltip component allows you to have help text or expanded descriptive information attached to a link element.

Example:
Some content for the informative tooltip.
With many lines of text, the tooltip box grows taller but the icon remains centered to the left of the text.
Hover Over Me Hover Over Me
Click Me
Usage:

Tooltips can be added to any <a> element. Add a rel="tooltip" and enter the title attribute which will become the tooltip content. The icon will be added automatically.

Tips appear by default on hover and by clicking. If clicked, you can dismiss the tip by clicking anywhere on the page or press ESC.

Disable hover by adding class="nohover" to the anchor tag.

By default, the click event on the anchor will be cancelled. If you need the anchor to be clickable, add class="clickthrough" to the anchor.

The tooltip will normally appear to the right side of the link but if it is too close to the edge of the screen, it will adapt to fit.

Variation: You can change the style from a tip to error by changing to rel="errortip".

Code Sample:
<a href="#" title="Hover for helpful information" rel="tooltip" >Hover Over Me</a>

<a href="#" title="Hover for helpful information" rel="tooltip" class="nohover">Click Me</a>

<a href="http://google.com/" title="You can search for websites here" rel="tooltip" class="clickthrough">Google.com</a>
