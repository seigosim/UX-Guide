Breadcrumbs

The Breadcrumbs component provides a list of links that represent the hierarchy of the current page.

Example:
  • Home
  • /
  • Web
  • /
  • Components
  • /
  • Breadcrumbs
Usage:

Breadcrumb links are built using an unordered list with the breadcrumbs class assigned. (<ul class="">)

Each link is placed within a <li> element. The separator characters are placed in their own <li> element with a separator class. You may use any text character for a separator.

The final element gets a n "active" class which changes its styling

Code Sample:
<ul class="breadcrumbs">
  <li><a href="index.htm">Home</a></li>
  <li class="separator">/</li>
  <li><a href="#">Web</a></li>
  <li class="separator">/</li>
  <li><a href="#">Components</a></li>
  <li class="separator">/</li>
  <li class="active"><a href="#">Breadcrumbs</a></li>
</ul>
