Tab Group

The Tab Group component is a compact UI element that can contain several "tabs" of content. The tabs titles can be clicked to show one tab body at a time without reloading the page.

Properties:
  • Minimum tab: 2
  • Maximum tabs: 5
  • The first tab will be open by default
Variations:

Columns As seen in the sample: class="tabgroup columns"

Rows Tabs along the left side: class="tabgroup rows"

Rows Right Tabs along the right side: class="tabgroup rows right"

Accordion Tabs titles are stacked vertically and the content expands down.

Usage:
  • Tab groups are built around the unordered list element (<ul>)
  • Each tab must contain one <a> element with class="title". Use href="javascript:void(0)"
  • Each tab must contain one element with class="body"
Code Sample:
<ul class="tabgroup columns">
 <li class="tabpos1 active">
  <a href="#" class="title">Tab Title</a>
  <div class="body" style="height:350px">
   Tab Content Here
  </div>
 </li>
 <li class="tabpos2">
  <a href="#" class="title">Tab Title</a>
  <div class="body" style="height:350px">
   Tab Content Here
  </div>
 </li>
</ul>

  </div>
 </li>
