Carousel

The Carousel component presents a series of banner combonents in a slideshow. Each image is clickable. After the last banner is shown, the carousel starts over with the first banenr.

Properties:
  • Minimum number of banners: 2
  • Maximum number of banners: 5
Example:
Brownie caramels gingerbread wypas wypas pie. Fruitcake cake toffee cake cupcake powder. Topping liquorice faworki tootsie roll jujubes applicake.
