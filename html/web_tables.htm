Tables

Best Practices

Use tables only to display tabular data. i.e. data must be aligned across all rows. If you are only looking to align elements side-by-side, use the grid layout styles instead.

Use table header (<th>) tags instead of applying background colors to table cells.

Define the width of your columns in the header or first row of the table only. Percentage widths are preferred.

Use only as many columns as you need for the data cells. Avoid adding extra "spacer" columns. Padding within the cell is almost always the better solution.

Avoid rowspan. This tends to create confusing table layouts. Rethink a clearer way to display the data.
