Video Tab

Example:
  • Videos
    Videos
    The Arbonne Opportunity We Are Arbonne Arbonne Essentials Pure Vibrance Arbonne Essentials - 30 Days to Fit The Millionaire Mom | Adventures in Arbonne The Roller Queen | Adventures in Arbonne
Usage:
  • Inside the tab body, add the video player containers as shown in the example code.
  • In the videoList container, add all the video links with the href attributes pointing to the YouTube page. Each link requires a class="youtubelink" to automatically insert the thumbnail image and to allow it to play inside the tab. Otherwise it will be an external link.
  • The video title is required for each of the video links.
  • The interface can support any number of videos and will scroll 3 thumbnails at a time.
Code Sample:
<ul  class="tabgroup columns" >
 <li class="videoTab">
  <a href="javascript:void(0)" class="title">Videos</a>
  <div class="body"  >
   <div class="videoContainer">
    <h5>Videos</h5>
     <div class="videoPlayer">
      <iframe id="YTFrame"  width="100%" height="100%" src="" frameborder="0" allowfullscreen></iframe>
     </div>
    </div>
    <div class="videoLinksPane">
     <a class="up arrow"></a>
     <div class="videoList">
      <a class="youtubeLink" href="http://www.youtube.com/watch?v=vAR6KRV2sIo" title="Watch Video">The Arbonne Opportunity</a>
     </div>
     <a class="down arrow"></a>
    </div>
   </div>	
 </li>									
</ul>
