Usage:

Create the modal content container anywhere in the document. Add a class="modal_content".

Add a height & width style to the modal content element.

The modal window will be automatically created using the contents of the container but the container itself will be discarded.

Any links with class="close_button" will be assigned the close dialog action. The dialog can also be dismissed by the close icon in the upper right corner or by clicking the background.

Insert the JQuery command inside a document.ready() block to create the modal.

To open the window via scripting, use: modal1.openWindow(). Close with modal1.closeWindow()

Modal Dialog: $('#window1').modalWindow($('#click1'),{'darken':true,'clickbg':true,'windowtype':'rounded','closeicon':true,'height':'300px','width':'500px'});
Where winodw1 is the ID of the modal content container and click1 is the ID of the link that opens the dialog.

Image Lightbox: $().modalWindow('#click2',{'darken':true,'imgsrc':'../img/shop_home_banner_FPO.jpg'});
Where click2 is the link that opens the image and the image source is specified in the options.

Options:
  • Darken background - darken: true (default true)
  • Click on background to close - clickbg: true (default true)
  • Rounded Corners - windowtype: 'rounded' (default is square)
  • No close icon in corner - closeicon: false (default true)
  • Specify Height - height: '300px' (default is to use modal content's size)
  • Specify Width - width: '300px' (default is to use modal content's size)
  • Image Source - imgsrc: '[URL]' (default none). This is used for lightbox style images
  • Remove padding - padding: false (default true). Remove padding if you want an image to stretch to the edges of the window.

By default the window is closable by clicking outside of the window. If you require forcing a confirmation in the dialog box, use these options: {'clickbg':false,'closeicon':false}. You can then use the scripting command to close the window after the action has completed.

Code Sample:
<script>
 $(document).ready(function(){
	var modal1 =$('#window1').modalWindow($('#click1'),{'windowtype':'rounded'});
 });
</script>

<a href="javascript:void(0)" id="click1">Click to open modal window</a>

<div id="window1" class="modal_contents" style="width: 500px;">
  <p>Modal window contents.</p>
  <div class="center-align"><a href="javascript:void(0)" class="close_button button">CLOSE</a></div>
</div>


You could also have the window pop open on page load by chaining the the openWindow() command:
var modal1 = $('#window1').modalWindow('#click1',{'windowtype':'rounded'}).openWindow();
